Hexo admin is really useful!

hexo admin

You don’t need to write your blog with vim or other Markdown app any more. It provides a admin page to write post/pages, you can deploy your site automatically. The most amazing thing is the function of pasting picture!

But I may still need to learn more about Markdown.
